## About project

Sends daily automatic news emails depends on the user's preferences based on the user's data provided

## Build with:
- [News API](https://newsapi.org/)
- [Yagmail](https://pypi.org/project/yagmail/)
- [Pandas](https://pandas.pydata.org/)
